# 資産管理ダッシュボード

家族（夫婦＋子供3人）のライフプラン可視化を見据えた、資産の現状把握と
将来シミュレーションのための Streamlit アプリです。Google スプレッドシートを
データストアとして使用します。

## 構成

| ファイル | 役割 |
| --- | --- |
| `app.py` | Streamlit 本体（入力フォーム＋ダッシュボード） |
| `pages/1_ライフプラン.py` | ライフプランシミュレーション画面（マルチページ） |
| `gsheet.py` | Google Sheets 認証・読み込み・追記・設定読込 |
| `analytics.py` | 集計／資産分類／複利シミュレーション（純粋関数） |
| `lifeplan.py` | 教育費・年次キャッシュフロー計算（純粋関数） |
| `config.py` | シート名・列名・資産分類キーワード・シミュレーション既定値 |
| `test_connection.py` | スプレッドシート接続テスト |
| `_smoke_analytics.py` | analytics の動作確認（API 不要） |
| `_smoke_lifeplan.py` | lifeplan の動作確認（API 不要） |
| `credentials.json` | サービスアカウント鍵（**Git 管理外**） |

## 主な機能

- **入力フォーム**: 『マスタ』シートの 名義／口座名／カテゴリ をドロップダウンで参照し、
  『データベース』シートに 1 行追記（APPEND）。
- **ポートフォリオ比較**: 年月を 2 つ選び、カテゴリ別比率を左右の円グラフで比較。
- **総資産推移＋複利シミュレーション**: 実績の折れ線に続けて、現在額と月平均増加額を
  起点に「年利（既定 5%）で運用した将来 10〜20 年」の予測ラインを点線で描画。
- **安全資産 vs リスク資産**: 最新月のデータをドーナツチャートで常時表示
  （分類は `config.py` のキーワードで判定）。
- **ライフプランシミュレーション**（サイドバー上部のページ切替から）:
  初期資産を起点に、以下を反映した高精度の年次シミュレーションを行います。
  - **収入**: 給与（現役中）／年金（受給開始年齢以降）／退職金（該当年に一時加算）
  - **リタイアと年金の分離**: 「リタイア年齢」と「年金受給開始年齢」を別設定でき、
    その間の「年金空白期間」は純取り崩しとして計算
  - **運用利回り**: リタイア前／後で年利を切替
  - **住居費（住宅ローン）**: 現在額 → 金利見直し後 → 完済で0円 の3段階
  - **生活費の分離**: 「基本生活費」＋子の年齢に応じた養育費の自動加算（傾斜）
  - **教育費**: 子ども3人×各ステージの公立/私立を個別選択
  - **多子世帯支援**: 児童手当（第3子以降増額）／大学無償化（扶養3人以上）
  - **カスタム・ライフイベント**: 任意の年齢に単発の収入/支出を追加（車買い替え等）

  結果は「資産推移（折れ線）＋年間支出の積み上げ棒＋収入線＋マイルストーン縦線」の
  複合グラフ、リタイア時予測資産・資産寿命のカード、年次キャッシュフロー詳細表に反映。
  生活費・教育費の初期値は『シミュレーション設定』シート（無ければ自動作成）から読み込むため、
  シートを書き換えれば「🔄 データを再読み込み」で自動反映されます。

## セットアップ（初回のみ）

事前準備として、Google Cloud 側で以下の 2 点が必要です。

1. **Google Sheets API を有効化**（プロジェクト `asset-app-500305`）
   https://console.cloud.google.com/apis/library/sheets.googleapis.com?project=asset-app-500305
2. **スプレッドシートを共有**: 対象シートを次のサービスアカウントに「編集者」で共有
   `asset-manager@asset-app-500305.iam.gserviceaccount.com`

依存ライブラリのインストール:

```powershell
& "$env:LOCALAPPDATA\Programs\Python\Python312\python.exe" -m pip install -r requirements.txt
```

> 注: pandas は Windows の Smart App Control 対応のため `2.2.3` に固定しています。

## 実行

接続テスト:

```powershell
& "$env:LOCALAPPDATA\Programs\Python\Python312\python.exe" test_connection.py
```

アプリ起動:

```powershell
./run_app.ps1
```

ブラウザで http://localhost:8501 が開きます。
